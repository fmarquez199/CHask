# **Compile and run**

1. Clone repo.

```sh
$> git clone https://github.com/fmarquez199/Playit.git
$> cd Playit
```

2. Compile and run.

```sh
stack run programs/filename.game
```

## Run unit tests

```
stack test
```

## Get stack here

https://docs.haskellstack.org/en/latest/install_and_upgrade/
